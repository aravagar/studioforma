# Studioforma

A premium brand and product website for Studioforma, a wood-veneer 
furniture manufacturer based in New Delhi, India. The site showcases 
the product range (doors, wall panels, and kitchen shutters), the 
company's manufacturing background, and gives architects, interior 
designers, builders, and homeowners a way to get in touch.

**Client:** Abhiroop (Director, Studioforma — a Realply Industries brand)

**Live site:** https://aravagar.github.io/studioforma

## Pages
- Home — hero, product highlights, brief about
- About Us — company story and background
- Catalogues — full door range and specifications
- Experiences — finishes and materials
- Factory — manufacturing capability
- Projects — work at scale
- Contact Us — inquiry form

## Built With
- Semantic HTML5
- External CSS 
- Responsive design with a mobile breakpoint
- Deployed on GitHub Pages

## What I Learned
Building for a real client meant the content had to come first, not the 
design. I used AI to generate the initial HTML structure from a hand drawn 
sketch and the product catalogue, then reviewed and corrected every section 
so it reflected real Studioforma content instead of placeholder text. I 
learned how to use CSS Grid with repeat(autofit, minmax) for the 
product layout, position, fixed for the vertical navbar, and media queries 
to collapse that navbar on mobile. The biggest lesson was that a clean, 
honest page built around what the client actually offers beats a flashy 
one full of invented detail.
